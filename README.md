# SQL Sales Dashboard

This project is a Sales Analytics Dashboard built using Python, Streamlit, and PostgreSQL.

It connects to a SQL database, retrieves sales data, and visualizes insights such as total revenue, sales by city, and top products.

## Technologies Used
- Python
- PostgreSQL
- Pandas
- Streamlit

## Features
- Live data from SQL database
- Revenue calculation
- Sales by city visualization
- Top products analysis

## How to Run
pip install -r requirements.txt
python -m streamlit run sql_dashboard.py
